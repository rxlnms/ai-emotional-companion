# UX Flow (MVP)

## Screen A — Chat
- Input + assistant reply
- Button: "Self-Awareness Session (30s)"

## Screen B — Consent modal
Text:
- Camera runs only during this session
- No video stored (default)
- You can stop anytime
Buttons:
- Start / Cancel

## Screen C — Camera session
- Timer 30s
- Stop button
- Minimal UI

## Screen D — Results
- “Possible signs of: tension/stress/calm/fatigue”
- Confidence (low/med/high)
- Question: “Want to explore what might be causing it?”
Options:
- Yes (Reflection)
- Not now (Back to chat)

## Screen E — Reflection
5–7 prompts, e.g.:
1) What happened today?
2) What is the strongest emotion right now?
3) Where do you feel it in the body?
4) What thought repeats?
5) One small action in next 10 minutes?
