# UX Flow — Web MVP

## Screen 1: Chat
Components:
- Message list
- Text input + send
- Button: “Self-Awareness Session (30s)”

## Screen 2: Consent Modal
Copy (example):
- “Camera will be used only for this 30-second session.”
- “By default, we do not store raw video.”
- “This tool is for self-reflection, not therapy or diagnosis.”
Buttons:
- Start session
- Cancel

## Screen 3: Camera Session (30s)
Components:
- Camera preview
- Timer (30s countdown)
- Stop button
Behavior:
- Auto-stop at 30s
- Immediately proceed to results

## Screen 4: Results
Components:
- “Possible signs of: Tension (Confidence: Medium)”
- 1–2 lines: “These signals can sometimes correlate with stress or fatigue.”
- CTA buttons:
  - “Explore what might be causing this”
  - “Not now”

## Screen 5: Reflection Flow (5–7 prompts)
Example prompts:
1) What happened today or right before this?
2) Where do you feel it in your body (if anywhere)?
3) What thought keeps looping?
4) What are you trying to control right now?
5) What do you need in the next 10 minutes?
6) One small action you can take now?
7) Write one supportive sentence to yourself.

## Screen 6: Summary
- Short recap (user’s words + one neutral reflection)
- Button: Back to chat
- Optional: “Clear session data”
